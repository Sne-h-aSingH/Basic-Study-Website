What is Visual Basic?

Introduction

  • Visual Basic is a highly programming language.It has several features that allows the programmer to deveop applications for MS-Windows in an easy and efficient manner.

  • Windows based application have a consistent useer interface, that helps the user to simply 'Point and Click'.

  • Visual Basic has two parts.The "Visual" part refers to the method used to create the graphical user interface(GUI).The "basic" part refers to the BASIC language.

Visual Basic

Visual Basic is an Event Driven programming language for creating application for Microsoft Windows.

VB has two components-:

  1. An entensive collection pf prewritten tools, called control in the form of icons,Some example of controls are command buttons, text boxes,option buttons, checkbox etc.
  2. A complete set of program commands, drived from BASIC programming language.

It is one of the first programming language which incoporates"What You See is What You Get",environment

Visual Basic is a tool that allows you to develop windows(GUI-Graphic User Interface)windows.

An event refers to occurence of a user-action, such as when the user click the mouse or when presses a key on the keyboard etc.

Visual Basic has features of Object Oreinted programming language.

Oject-related concept in VB-:

  1. Forms-: In VB a window is called a form. All application in VB at least one form.You can place various controls on the form. Each form contains a title bar as shownin figure-
  2. controls-: The icons with which the user interact are called controls.
  3. Objects-: Forms and controls are referred to collectively as objects.
  4. Properties-: A property is a characteristics of an object such as its size,its caption or its colour etc.
  5. Methods-: A method in VB is a predefined action that operates on a specific object(i.e. form or contorl.)For example, Set Focus method is to move the focus to the specified object.

Hardware and Software requirement of Visual Basic

Hardware and Software requirement of VB are-:

  1. Microsoft Windows 95 or later.
  2. 486DX/66 MHz or higher processor
  3. A Mouse
  4. A CD-ROM drive or DVD drive
  5. Minimum 16 MB of RAM for windows 95, but 32 MB of RAM is desirable.

Setting up Visual Basic

To set up VB from CD,the following steps are used:
